# Contributing

## Requirements

- Node
- NPM
- Git

## Clone

```bash
git clone https://github.com/aminnairi/react-table
cd react-table
```

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Types

```bash
npm run types
```

## Publish

```bash
npm publish --access public
```
